## 1. 产品概述

库存补货协同看板是一款面向仓储运营和供应链管理人员的实时监控与协同工具，旨在将仓库库存状态、SKU 风险等级、补货建议、供应商协同状态和预警信息统一呈现在一个看板中，实现"风险→建议→供应商→行动"的闭环处理。

- 解决库存断货、补货延迟、供应商信息缺失等供应链痛点
- 目标用户：仓库主管、采购经理、供应链分析师

## 2. 核心功能

### 2.1 用户角色

| 角色 | 核心权限 |
|------|----------|
| 仓库主管 | 查看所有仓库库存和预警，确认补货建议 |
| 采购经理 | 查看供应商协同状态，发起补货订单 |
| 供应链分析师 | 查看趋势概览，分析预警数据 |

### 2.2 功能模块

1. **看板主页面**：仓库列表概览、SKU 风险面板、补货建议列表、供应商协同状态、趋势图表、预警模拟

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 看板主页面 | 仓库列表 | 展示所有仓库及其库存汇总（总SKU数、风险SKU数、缺货数），点击可筛选该仓库下的SKU风险 |
| 看板主页面 | SKU 风险面板 | 按"断货/极低/偏低/正常"四级风险展示SKU，库存为0标红高亮，显示处理逻辑标签（如"需紧急补货""供应商缺失""预警过多"） |
| 看板主页面 | 补货建议 | 根据SKU风险自动生成补货建议（建议补货量、目标库存、优先级），联动供应商状态，供应商缺失时标记"需确认供应商" |
| 看板主页面 | 供应商协同 | 展示供应商列表及状态（在线/离线/响应中），显示供应商响应率和交货周期，缺失供应商的SKU高亮 |
| 看板主页面 | 趋势概览 | 近7/30天库存趋势折线图、预警数量趋势图、补货完成率趋势图 |
| 看板主页面 | 本地模拟预警 | 模拟预警推送（断货预警、库存低于安全线、供应商延迟），可手动触发预警场景，展示预警处理流程 |

## 3. 核心流程

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深靛蓝(#1e293b) + 琥珀色警示(#f59e0b) + 翡翠绿安全(#10b981) + 玫红危险(#ef4444)
- 按钮风格：圆角(8px)，风险操作用实色填充，常规操作用描边
- 字体：DM Sans（标题）+ Source Sans 3（正文），标题16-24px，正文12-14px
- 布局风格：顶栏导航 + 左侧仓库筛选 + 中间主内容区多面板网格
- 图标风格：Lucide 线性图标

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 看板主页面 | 顶栏导航 | 品牌 Logo、页面标题、预警铃铛(带未读数)、时间显示 |
| 看板主页面 | 仓库列表(左侧) | 仓库卡片列表(名称+库存摘要)，选中高亮，风险仓库标红边框 |
| 看板主页面 | SKU 风险面板 | 分色标签页(断货/极低/偏低/正常)，SKU表格(名称/库存/安全线/风险/处理标签)，库存为0行红色背景 |
| 看板主页面 | 补货建议 | 建议卡片列表(SKU名/当前库存/建议量/优先级/供应商)，供应商缺失时橙色警告标签 |
| 看板主页面 | 供应商协同 | 供应商状态卡片(名称/状态灯/响应率/交货周期)，缺失供应商关联SKU列表 |
| 看板主页面 | 趋势概览 | 折线图(库存趋势)、柱状图(预警趋势)、环形图(补货完成率) |
| 看板主页面 | 本地模拟预警 | 预警场景按钮组、预警消息列表(时间/级别/内容/处理状态)、处理流程时间线 |

### 4.3 响应式

- 桌面优先设计，1920px为主视口
- 左侧仓库列表可折叠
- 面板网格在小屏自动堆叠
